# Декодирование сообщения

## Суть задачи

Есть строка из цифр. Каждая русская буква кодируется своим номером от `1` до `33`, а пробел кодируется `0`.

Нужно не восстановить все тексты, а только посчитать, сколько различных декодирований возможно.

## Реализация

Основная функция:

```python
message(q: str) -> int
```

Файл решения: `message.py`

## Как работает алгоритм

Используется динамическое программирование по префиксам строки.

Массив `drt[i]` хранит количество способов расшифровать первые `i` символов строки.

На каждом шаге проверяются три случая:

1. Последний символ — цифра от `1` до `9`, значит можно продолжить все декодирования префикса `i - 1`.
2. Последние два символа образуют число от `10` до `33`, значит можно добавить все декодирования префикса `i - 2`.
3. Если текущий символ равен `0`, он интерпретируется как пробел.

## Какие алгоритмы применены

- динамическое программирование
- подсчёт количества путей по строке

## Сложность

- по времени: `O(N)`
- по памяти: `O(N)`

## Пример использования

```python
from message import message

print(message("21705"))  # 3
```

## Комментарий по реализации

Функция не строит сами варианты сообщений, а только считает их количество. Это позволяет удержать линейную сложность даже на длинных входных строках.
