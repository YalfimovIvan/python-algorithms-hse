# Транскрипция РНК

## Суть задачи

Нужно преобразовать строку ДНК в строку РНК по правилам:

- `A -> U`
- `C -> G`
- `G -> C`
- `T -> A`

## Реализация

Основная функция:

```python
to_rna(dna_strand)
```

Файл решения: `rna_transcription.py`

## Как работает алгоритм

В данной реализации используется цепочка строковых замен.

Чтобы не потерять исходные символы при взаимной замене `G -> C` и `C -> G`, применяется промежуточный маркер:

1. `A` заменяется на `U`
2. `G` временно заменяется на `g`
3. `C` заменяется на `G`
4. временный `g` заменяется на `C`
5. `T` заменяется на `A`

Этот трюк позволяет корректно выполнить взаимные замены без словаря и посимвольного прохода.

## Сложность

Логически задача линейная относительно длины строки, хотя технически в коде выполняется несколько проходов через `.replace()`.

- по времени: `O(N)`
- по памяти: `O(N)`

## Пример использования

```python
from rna_transcription import to_rna

print(to_rna("AGCTAT"))  # UCGAUA
```
