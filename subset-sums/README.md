# Различные суммы подмножеств

## Суть задачи

Дан массив целых чисел. Нужно определить, сколько различных значений можно получить, если брать суммы всех возможных подмножеств.

Пустое подмножество тоже учитывается, поэтому сумма `0` всегда входит в ответ.

## Реализация

Основная функция:

```python
sums(array: list) -> int
```

Файл решения: `sums.py`

## Как работает алгоритм

Используется динамика по множеству достижимых сумм.

Идея:

1. Изначально достижима только сумма `0`.
2. Для каждого числа `num` берутся все суммы, которые уже достижимы.
3. К каждой такой сумме добавляется `num`.
4. Новые суммы объединяются с уже найденными.

В коде это выражено через множества:

- `sums` — уже достижимые суммы
- `new_sums` — суммы, которые появляются после добавления нового элемента

## Какие алгоритмы применены

- динамическое программирование по множеству состояний
- использование `set` для автоматического удаления повторов

## Сложность

В худшем случае число достижимых сумм экспоненциально, поэтому:

- по времени: до `O(2^N)`
- по памяти: до `O(2^N)`

На практике решение часто работает заметно лучше, если много сумм совпадает.

## Пример использования

```python
from sums import sums

print(sums([1, 1, 2]))  # 5
```
