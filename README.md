# Алгоритмы на Python

Сборник небольших алгоритмических задач, решённых на Python. Каждая папка содержит отдельную постановку, решение и тесты.

## Содержимое

- `big-politics` — оптимальная стоимость последовательного объединения
- `closest-pair-of-points` — поиск ближайшей пары точек
- `k-closest-elements` — `k` ближайших элементов в отсортированном массиве
- `message-decoding` — подсчёт числа допустимых расшифровок цифровой строки
- `raindrops` — простая задача на делимость
- `rna-transcription` — транскрипция ДНК в РНК
- `run-length-encoding` — кодирование и декодирование длин серий
- `subset-sums` — количество различных сумм подмножеств

## Какие алгоритмы используются

- `big-politics` (жадный алгоритм на минимальной куче)
- `closest-pair-of-points` (рекурсивное разделяй и властвуй)
- `k-closest-elements` (бинарный поиск + расширение окна двумя указателями)
- `message-decoding` (динамическое программирование по префиксам)
- `subset-sums` (динамика через множество достижимых сумм)


### Запуск всех тестов из корня репозитория

```bash
pytest
```

### Запуск отдельной задачи

```bash
cd big-politics
pytest
```
