# K ближайших чисел

## Суть задачи

Дан отсортированный массив и число `target`. Нужно вернуть `count` элементов, которые ближе всего к `target`, не нарушая исходный порядок элементов в массиве.

## Реализация

Основная функция:

```python
closest(array: list, target: int, count: int) -> list
```

Файл с решением: `closest.py`

## Как работает алгоритм

Решение состоит из двух этапов:

1. Бинарным поиском находится позиция, где расположен `target` или куда он должен был бы быть вставлен.
2. Затем окно ответа расширяется двумя указателями:
   - левый идёт влево
   - правый идёт вправо
   - на каждом шаге выбирается элемент, который ближе к `target`

Итоговый ответ возвращается как срез:

```python
array[left + 1:right]
```

## Какие алгоритмы применены

- бинарный поиск
- техника двух указателей
- работа с полуинтервалом ответа

## Сложность

- поиск стартовой позиции: `O(log N)`
- расширение окна: `O(k)`
- итоговая сложность: `O(log N + k)`

## Ограничения

- входной массив должен быть уже отсортирован
- функция не занимается сортировкой сама

## Пример использования

```python
from closest import closest

print(closest([1, 4, 8, 10], 2, 3))  # [1, 4, 8]
```
