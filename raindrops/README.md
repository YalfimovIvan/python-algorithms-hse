# Капли

## Суть задачи

Это вариация классической задачи FizzBuzz.

Для числа нужно вернуть строку по таким правилам:
- если число делится на `3`, добавить `Pling`
- если число делится на `5`, добавить `Plang`
- если число делится на `7`, добавить `Plong`
- если ни одно условие не выполнено, вернуть само число как строку

## Реализация

Основная функция:

```python
convert(number)
```

Файл решения: `raindrops.py`

## Как работает алгоритм

Алгоритм очень прямолинейный:

1. Создаётся пустая строка.
2. Для каждого делителя проверяется остаток от деления.
3. Если условие выполнено, в строку дописывается соответствующее слово.
4. Если строка осталась пустой, возвращается исходное число.

## Сложность

- по времени: `O(1)`
- по памяти: `O(1)`

## Пример использования

```python
from raindrops import convert

print(convert(28))  # Plong
print(convert(30))  # PlingPlang
print(convert(34))  # 34
```
