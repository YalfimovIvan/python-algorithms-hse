# Кодирование длин серий

## Суть задачи

Нужно реализовать упрощённый вариант Run-Length Encoding:
- серия одинаковых символов заменяется на `число + символ`
- если символ встречается один раз, число `1` не записывается

Пример:

```text
AAAABCC -> 4AB2C
```

## Реализация

Файл решения: `run_length_encoding.py`

Основные функции:

```python
encode(s)
decode(s)
```

## Как работает `encode`

Алгоритм проходит строку слева направо и считает длину текущей серии:

1. Если следующий символ совпадает с текущим, счётчик увеличивается.
2. Если серия заканчивается:
   - при длине больше `1` в результат добавляется число и символ
   - иначе добавляется только символ

## Как работает `decode`

Алгоритм также идёт слева направо:

1. Если встречаются цифры, они накапливаются в строковый счётчик.
2. Когда встречается буква, она повторяется нужное число раз.
3. Если счётчик пустой, символ просто добавляется один раз.

## Какие алгоритмы применены

- линейный проход по строке
- счётчик текущей серии
- разбор числа произвольной длины в декодере

## Сложность

- `encode`: `O(N)`
- `decode`: `O(N)`

## Ограничения

Как и в классической постановке задачи, вход для кодирования предполагается строкой без цифр, иначе декодирование станет неоднозначным.

## Пример использования

```python
from run_length_encoding import encode, decode

encoded = encode("WWWWBBBW")
print(encoded)
print(decode(encoded))
```
